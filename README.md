# Jira Jumper

A Chrome extension to quickly navigate to PGNG tickets.

## How to Install
1. Download the `jira-jumper.zip` from the [Releases](#) page.
2. Unzip the folder.
3. Open Chrome and go to `chrome://extensions/`.
4. Enable **Developer Mode** (top right).
5. Click **Load Unpacked** and select the unzipped folder.
